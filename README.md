# v3-template-h5

#### 介绍

基于 vue3+vite+vant+pinia+tailwindcss 的 webapp 模板

#### 安装教程

1.  git clone https://github.com/OGweixin007/v3-template-h5.git
2.  cd v3-template-h5
3.  pnpm install
4.  pnpm run dev

#### 参与贡献

1.  Fork 本仓库
2.  新建 Feat_xxx 分支
3.  提交代码
4.  新建 Pull Request
